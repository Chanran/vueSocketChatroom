## 安装依赖

1. node7.x

2. 自动重启node服务工具

```
npm i -g nodemon
```

3. QA(代码质量保证) 工具

    给你的编辑器装一个eslint插件(比如vscode的"eslint")

## 下载

``` bash
# 克隆
git clone git@github.com:Chanran/vueSocketChatroom.git
cd vueSocketChatroom

# 安装依赖
npm install -d
```

## 启动

```
npm run dev
npm run server # 另开一个终端
```

访问 [http://localhost:8080/](http://localhost:8080/)

## 部署

```
npm install -g pm2 # 只安装一次
npm i -d --production
npm run build
npm run deploy
```
